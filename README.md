#  Truck Company Project

A full-stack interactive web application designed to streamline truck rentals, branch activity tracking, and customer requests.

## Features
* Interactive Booking: Real-time truck availability and reservation system.
* Database Integration: Secure storage for branches data and request history.
* Responsive Design: Fully functional on desktop.

## Tech Stack
* Frontend: JavaScript, HTML, CSS
* Backend: Node.js, Express
* Database: SQL

## Installation
1. Clone the repo: `git clone https://github.com/NicolasSlat/truckcompany-project`
2. Install dependencies: `npm install`
3. Set up the database using the provided `database.sql` file.
4. Start the server: `node server.js`
